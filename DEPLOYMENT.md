# 阿里开放平台 API 申请 - 部署说明

## 一、项目部署到 Vercel

### 1. 推送代码到 GitHub

```bash
cd /workspace/projects
git add .
git commit -m "feat: 初始化闲鱼数据分析平台"
git branch -M main
git remote add origin <你的GitHub仓库地址>
git push -u origin main
```

### 2. 在 Vercel 部署

1. 访问 [Vercel](https://vercel.com)
2. 点击 "New Project"
3. 导入你的 GitHub 仓库
4. 点击 "Deploy"

部署完成后，Vercel 会提供一个公开的访问地址，例如：`https://your-project.vercel.app`

## 二、API 申请材料准备

### 1. MRD（市场需求文档）

**项目名称**: 闲鱼商品数据分析平台

**业务背景**:
闲鱼作为国内领先的二手交易平台，日活用户数千万，商品种类丰富。商家在闲鱼上经营面临以下痛点：
- 无法实时监控同类商品价格变化
- 定价依赖个人经验，缺乏数据支撑
- 难以全面分析竞争对手策略
- 缺乏历史成交数据参考

**目标用户**:
- 闲鱼个人卖家
- 闲鱼小店商家
- 二手商品经营者

**核心需求**:
1. 实时价格监控：自动跟踪同类商品价格走势
2. 智能定价建议：基于数据提供最优价格推荐
3. 竞品分析：分析竞争对手的销售策略
4. 数据可视化：直观展示市场趋势

**预期收益**:
- 帮助商家提升商品竞争力
- 优化定价策略，提高成交率
- 降低市场调研成本

---

### 2. PRD（产品需求文档）

**系统名称**: 闲鱼商品数据分析平台

**技术架构**:
- 前端: Next.js 16 + React 19 + TypeScript
- 样式: Tailwind CSS
- 部署: Vercel
- 后端: 阿里开放平台 API

**核心功能模块**:

#### 2.1 用户管理模块
- 用户注册/登录
- 账户信息管理
- 权限控制

#### 2.2 商品监控模块
- 关键词搜索商品
- 收藏关注商品
- 价格变动提醒

#### 2.3 数据分析模块
- 价格走势图表
- 成交量统计
- 热门商品排行

#### 2.4 智能定价模块
- 基于历史数据的定价建议
- 同类商品价格对比
- 竞品价格分析

**API 使用计划**:

| 接口名称 | 接口用途 | 调用频率 |
|---------|---------|---------|
| 商品搜索接口 | 根据关键词搜索闲鱼商品 | 按需调用 |
| 商品详情接口 | 获取商品详细信息及历史价格 | 按需调用 |
| 用户信息接口 | 获取卖家信用等级 | 按需调用 |
| 交易数据接口 | 获取历史成交记录 | 按需调用 |

**数据安全**:
- 用户数据加密存储
- API 访问权限控制
- 遵守阿里开放平台安全规范
